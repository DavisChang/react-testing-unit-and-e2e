## Start project
```
$ yarn dev:start
```

## Start unit testing

```
$ yarn run test
```

## Start e2e testing
You can see coverage report in index.html within your browsers

```
$ yarn run test:e2e:open
$ yarn run test:e2e:report
$ open ./coverage/lcov-report/index.html
```
